**1. Explique, com suas palavras, o que é machine learning?**  
Machine Learning (ou aprendizagem de máquina) é a habilidade de computadores aprenderem com dados e experiências ao utilizar algoritmos para identificar padrões e, consequentemente, poder prever ações e tomar decisões, sendo assim parte da Inteligência Artificial.


**2. Explique o conceito de conjunto de treinamento, conjunto de validação e conjunto de teste em machine learning.**  
- Conjunto de treinamento, como diz o nome, é usado para treinar o modelo.  
- Conjunto de validação é usado para comparar o desempenho de diferentes modelos e hiperparâmetros.  
- Conjunto de teste é usado para comprovar a generalização do modelo e verificar se ele funciona; são dados nunca vistos pelo modelo.  


**3. Explique como você lidaria com dados ausentes em um conjunto de dados de treinamento.**  
Inicialmente, tentaria identificar um padrão de ausência e a quantidade para avaliar possível exclusão da variável. Em seguida, para cada tipo de dado eu teria uma abordagem diferente. Por exemplo, para variáveis categóricas, faria imputação pela moda; já para variáveis numéricas, usaria a mediana.  


**4. O que é uma matriz de confusão e como ela é usada para avaliar o desempenho de um modelo preditivo?**  
A matriz de confusão é uma tabela que resume o desempenho de modelos classificadores, mostrando quantas previsões foram corretas e incorretas em cada classe, para que o modelo possa ser ajustado e tenha os classificadores melhorados em problemas reais. Ela é usada para calcular métricas que permitem não apenas identificar o desempenho geral, mas também os tipos de erros que o modelo comete.  


**5. Em quais áreas (tais como construção civil, agricultura, saúde, manufatura, entre outras) você acha mais interessante aplicar algoritmos de machine learning?**  
Gosto bastante das aplicações na área da saúde, pois acredito que é uma das principais em que conseguimos ver aplicação de perto, tanto na questão da gestão de unidades de saúde quanto em diagnósticos antecipados. As aplicações são múltiplas e todos saem ganhando.
